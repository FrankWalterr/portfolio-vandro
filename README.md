# Vandro Correia — Portfólio

Portfólio pessoal de Vandro Correia, Desenvolvedor de Software.

## Stack

- React 18 + Vite
- TailwindCSS (design tokens customizados)
- Framer Motion
- Lucide React

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy (Vercel)

1. Conectar repositório à Vercel
2. Framework preset: Vite
3. Deploy automático — subdomínio sugerido: `vandro-correia.vercel.app`
